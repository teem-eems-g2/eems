## Core Prototype - Phase 4 Implementation

### Quick Setup (Local Development)

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/teem-eems-g2/eems.git](https://github.com/teem-eems-g2/eems.git)
   cd eems
```
2. **Backend Setup:**
   ```bash
   cd backend
   npm install
   npm start
   ```
   Server runs on: http://localhost:5000

3. **Frontend Setup:**
   ```bash
   cd frontend
   npm install  
   npm start
   ```
   App runs on: http://localhost:3000

4. **Database Setup:**
   # Run in PostgreSQL
   ```bash
psql -U postgres -f database/schema.sql
```cha
